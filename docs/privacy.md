# Privacy Policy

**safe-read** is an open-source Rust library that runs entirely on your own
machine.

- **No data collection.** The library does not transmit, upload, or share any
  data. It only reads the byte slices you pass to it, in memory.
- **No telemetry.** No analytics, crash reporting, phone-home, or network
  activity of any kind. `#![no_std]`, no dependencies.
- **No accounts.** No registration, login, or API key.

This policy applies to the library as distributed via crates.io and GitHub. It
does not cover forks or modified copies.
